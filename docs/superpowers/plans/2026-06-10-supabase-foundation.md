# Supabase Foundation (Plan 2a) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create the Supabase project and database schema, and migrate the public storefront from hardcoded placeholder data (`lib/products.ts`, `lib/store-settings.ts`) to data read from Supabase.

**Architecture:** A single Supabase Postgres project hosts four tables (`products`, `store_settings`, `orders`, `reviews`) with row-level security allowing public read access to products, store settings, and approved reviews. `app/page.tsx` becomes an async Server Component that fetches products and store settings via a small Supabase client wrapper, then passes the data to a new `components/HomeClient.tsx` client component that contains all the existing interactive UI (tabs, cart, checkout). `orders` and `reviews` tables are created now (with RLS policies) but not yet used by the app — they're wired up in Plans 2b and 2c.

**Tech Stack:** `@supabase/supabase-js`, Next.js App Router (Server + Client Components), Vitest for unit tests of data-mapping functions (with a mocked Supabase client).

**Out of scope (future plans):** saving orders to the `orders` table (Plan 2b), admin panel and Supabase Auth (Plan 2c).

```sql
  whatsapp_target text not null,

  tabs_config jsonb not null,
```

```ts
export function createSupabaseClient(): SupabaseClient {

  const anonKey = process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY;
```

```ts
import { SupabaseClient } from "@supabase/supabase-js";
import { Product } from "./types";

interface ProductRow {
  id: string;
  name: string;
  description: string;
  price: number;
  image_url: string;
  category: Product["category"];
  badge: Product["badge"];
  stock_today: number | null;
  active: boolean;
}

export async function getProducts(client: SupabaseClient): Promise<Product[]> {
  const { data, error } = await client
    .from("products")
    .select("*")
    .eq("active", true)
    .order("category");

  if (error) {
    throw new Error(`Failed to load products: ${error.message}`);
  }

  return (data as ProductRow[]).map((row) => ({
    id: row.id,
    name: row.name,
    description: row.description,
    price: row.price,
    imageUrl: row.image_url,
    category: row.category,
    badge: row.badge,
    stockToday: row.stock_today,
    active: row.active,
  }));
}
```

```ts
import { SupabaseClient } from "@supabase/supabase-js";
import { StoreSettings } from "./types";

interface StoreSettingsRow {
  is_open: boolean;
  prep_time_minutes: number;
  min_order_value: number;
  hero_text: string;
  whatsapp_target: string;
  payment_methods: StoreSettings["paymentMethods"];
  delivery_options: StoreSettings["deliveryOptions"];
  tabs_config: StoreSettings["tabs"];
}

export async function getStoreSettings(client: SupabaseClient): Promise<StoreSettings> {
  const { data, error } = await client.from("store_settings").select("*").single();

  if (error) {
    throw new Error(`Failed to load store settings: ${error.message}`);
  }

  const row = data as StoreSettingsRow;

  return {
    isOpen: row.is_open,
    prepTimeMinutes: row.prep_time_minutes,
    minOrderValue: row.min_order_value,
    heroText: row.hero_text,
    whatsappTarget: row.whatsapp_target,
    paymentMethods: row.payment_methods,
    deliveryOptions: row.delivery_options,
    tabs: row.tabs_config,
  };
}
```

- [ ] **Step 1: Create `components/HomeClient.tsx`**

This is the existing interactive UI from `app/page.tsx`, now receiving data as props instead of importing it directly:

```tsx
"use client";

import { useState } from "react";
import { Header } from "@/components/Header";
import { InfoBar } from "@/components/InfoBar";
import { Hero } from "@/components/Hero";
import { Tabs } from "@/components/Tabs";
import { ProductGrid } from "@/components/ProductGrid";
import { CartBar } from "@/components/CartBar";
import { CheckoutModal } from "@/components/CheckoutModal";
import { Product, StoreSettings } from "@/lib/types";
import { useCart } from "@/hooks/useCart";

const TAB_TO_CATEGORY: Record<string, string> = {
  destaques: "destaque",
  cookies: "cookie",
  especiais: "especial",
};

export function HomeClient({
  products,
  storeSettings,
}: {
  products: Product[];
  storeSettings: StoreSettings;
}) {
  const [activeTab, setActiveTab] = useState(storeSettings.tabs[0].id);
  const [checkoutOpen, setCheckoutOpen] = useState(false);
  const { cart, total, add, remove } = useCart();

  const category = TAB_TO_CATEGORY[activeTab];
  const visibleProducts = products.filter(
    (product) => product.active && product.category === category
  );

  return (
    <main className="max-w-md mx-auto pb-4">
      <div className="deli-stripe" />
      <Header />
      <InfoBar settings={storeSettings} />
      <Hero text={storeSettings.heroText} />
      <Tabs tabs={storeSettings.tabs} activeTab={activeTab} onChange={setActiveTab} />

      {category ? (
        <ProductGrid
          products={visibleProducts}
          cart={cart}
          onAdd={add}
          onRemove={remove}
        />
      ) : (
        <div className="px-6 py-8 text-sm text-taupe">
          Conteúdo desta seção em breve.
        </div>
      )}

      <CartBar cart={cart} total={total} onReview={() => setCheckoutOpen(true)} />

      {checkoutOpen && (
        <CheckoutModal
          cart={cart}
          total={total}
          settings={storeSettings}
          onClose={() => setCheckoutOpen(false)}
        />
      )}
    </main>
  );
}
```
